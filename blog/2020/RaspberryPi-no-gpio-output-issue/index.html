Raspberry Pi 4B no GPIO output in C program: Solved!

June 9, 2020

I am using the latest Raspberry Pi 4B board and programming using C. It is found that when using the WiringPi library to control the GPIO output (digital/pwm), there is no actual voltage output while the program can just compile and run. On the other hand, the GPIO works normally if program with Python using the RPi.GPIO module.

The problem can be solved by updating the WiringPi to 2.52. You can not automatically get this version using the apt-get command as follows as it is not an official release yet.
